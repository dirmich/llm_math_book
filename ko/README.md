# LLM Math Book Exercises

직접 실행해 볼 수 있는 공개 실습 및 실험 코드입니다.

언어: [영어](../en/), [일본어](../jp/)

## 구성

- `notebooks/`: 한국어 실행 노트북
- `solutions/`: 한국어 해설 노트북
- `benchmarks/`: 한국어 CPU/GPU 성능 비교 스크립트
- `../src/llm_math/`: 공통 유틸리티
- `../data/`: 실습용 작은 데이터

## 빠른 시작

```bash
pip install -r requirements.txt
pip install -e .
jupyter lab
```

Colab에서는:

```bash
bash colab_setup.sh
```
