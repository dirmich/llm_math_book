# LLM Math Book Exercises

Public hands-on exercises and experiment code.

Languages: [Korean](../ko/), [Japanese](../jp/)

## Contents

- `notebooks/`: English runnable notebooks
- `solutions/`: English solution notebooks
- `benchmarks/`: English CPU/GPU benchmark scripts
- `../src/llm_math/`: shared utilities
- `../data/`: small practice datasets

## Quick Start

```bash
pip install -r requirements.txt
pip install -e .
jupyter lab
```

For Colab:

```bash
bash colab_setup.sh
```
